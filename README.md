# 📝 To-Do List App

A simple and efficient To-Do List web application designed to help users manage daily tasks easily. Built with modern web technologies, this project provides a clean interface and smooth user experience for organizing and tracking tasks.

---

## 🚀 Features

- 📝 Add new tasks  
- ✔️ Mark tasks as completed  
- 🗑️ Delete tasks  
- 🎨 Clean and minimal UI  
- ⚡ Lightweight and fast  
- 📱 Responsive design  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript

## 🎮 How It Works

1. Open the application  
2. Add tasks using the input field  
3. Mark tasks as completed  
4. Delete tasks when finished
